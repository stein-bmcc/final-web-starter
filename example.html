Examples of How to Add Media

How to add images, audio and video to a web page.

This page is an example page. You can copy from it to help you with creating your own pages.

Instructions

To add your own audio you will need to:

  • Upload your image files to the images folder.
  • Copy the figure element above including the img and figcaption elements. Or you can just write them on your own.
  • Change the src attribute in the audio element.

Adding in a Video File

Instructions

To use this template you will need to:

  • Upload your video file to the media folder.
  • Change the src attribute in the source element. Also change the type if your video is not mp4
  • Change the figcaption content to fit your video. If you made it then give yourself credit.
